# PixelFocus 2.0 — Study Space

Пиксельный сайт для учёбы и работы с атмосферными сценами, таймером Помодоро,
плейлистом и звуковым менеджером.

---

## 🚀 Быстрый старт (GitHub Pages)

1. Загрузи папку `pixelfocus/` в репозиторий GitHub
2. В настройках → Pages → выбери ветку `main`, папку `/ (root)`
3. Сайт будет доступен на `https://твой-ник.github.io/репозиторий/`

---

## 🎵 Как добавить звуки

### Способ 1 — через скрипт-установщик (рекомендуется)

```bash
pip install flask
python setup.py
# Открой http://localhost:5000
# Загрузи файлы через браузер
# Нажми "Готово" — скрипт самоудалится, playlist.json обновится
```

### Способ 2 — вручную

Положи файлы в нужные папки:

```
audio/
├── ambient/
│   ├── rain.mp3      ← или .wav / .ogg
│   ├── fire.mp3
│   ├── wind.mp3
│   ├── cafe.mp3
│   ├── thunder.mp3        ← фоновый гул грозы (петля)
│   ├── thunderstrike.mp3  ← удар грома при вспышке молнии
│   └── typing.mp3    ← фоновый звук клавиатуры
└── music/
    ├── lofi-01.mp3   ← любые имена
    ├── lofi-02.mp3
    └── ...
```

Затем обнови `playlist.json` вручную:

```json
{
  "tracks": [
    { "file": "lofi-01.mp3", "title": "Morning Rain" },
    { "file": "lofi-02.mp3", "title": "Coffee Shop" }
  ]
}
```

## 🎮 Управление

| Действие | Способ |
|---|---|
| Play / Pause музыку | Кнопка ▶ или `Space` |
| Следующий трек | Кнопка ⏭ или `Alt + →` |
| Предыдущий трек | Кнопка ⏮ или `Alt + ←` |
| Режим фокуса | Кнопка 👁 — **включается и выключается только кнопкой** |
| Выход из фокуса | Кнопка "✕ ВЫЙТИ ИЗ ФОКУСА" |
| Перетащить панель | Захвати за заголовок мышью |
| Свернуть панель | Кнопка — в правом углу заголовка |

## ⏱ Таймер Помодоро

- **25м** — стандартный рабочий блок
- **50м / 90м** — длинные сессии
- **✏** — произвольное время
- После каждого томата автоматический перерыв 5 минут
- 📊 Статистика: томаты, минуты фокуса, задачи

## 📁 Структура файлов

```
pixelfocus/
├── index.html          ← главная страница
├── playlist.json       ← список треков (авто-генерируется setup.py)

│   └── style.css
├── js/
│   ├── canvas.js       ← пиксельная графика

│   ├── player.js       ← музыкальный плеер
│   ├── timer.js        ← таймер
│   ├── tasks.js        ← список задач
│   ├── ui.js           ← интерфейс (drag, фокус, сцены)
│   └── main.js         ← инициализация
└── audio/

    └── music/          ← плейлист (любое кол-во MP3)
```

## 💡 Советы

- Если аудио-файл не найден, используется процедурная генерация звука (встроено в браузер)
- Для лучшего опыта используй петлевые (looping) записи без пауз
- Несколько атмосферных звуков можно включать одновременно
- Панели можно перетаскивать и сворачивать — позиции не сохраняются между сессиями (добавь localStorage если нужно)
